This Web-based information system, is designed to view, update, and delete records for a given item. You should use this system to maintain complete information on all records.

This system is built on the .Net platform using various technologies for various layers as given below

User Interface: AngularJS and BootStrap

Service Layer: Asp.Net Web API

Business Logic Layer: C#.Net

Data Access Layer: Entity Framework Code First

Back End: MS SQL Server 2014

Tools: Visual Studio 2017 and SQL Server Management Studio 2014
